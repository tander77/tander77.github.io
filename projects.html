My Projects

Here are some of the projects I have undertaken.

Hyperloop Pod Test Rig

Status: Completed

My capstone project in my final year of university was to design a test rig to evaluate the feasibility of a new levitation technology invented by the University of Toronto Hyperloop Team.

I was the project manager for my team, and as such, I laid out a schedule for tem members to complete their individual contributions.

My most interesting contribution to the design was a robot which would move a magnetic field sensor. For this, I hooked up an Arduino to a Raspberry Pi, and programmed the control system for this.

The attached image is of the wiring of the Arduino to two stepper motors as part of my prototyping.

From my experience being a project manager for my team, I have learned how to manage multiple people's schedules in a way that minimizes conflict and ensures that assigned tasks are completed.

This project has also solidified my knowledge of Arduino programming and wiring as well as serial communications between Arduino and Raspberry Pi.

Recipe Database

Status: Completed

As part of my software engineering course, we had a group project that tasked us with designing a website that would allow users to search for recipes based on the ingredients they have on hand.

I was responsible for meshing the front-end and back-end, however I ended up creating many front-end features and back-end functionalities.

The website was created using Python with the Flask microframework. We also used SQLAlchemy library for easier database manipulation. I implemented the functionality enabling users to add new ingredients and recipes, as well as the ability to view recipes.

I also implemented a unit conversion so that users may read ingredient quantities in their units of choice.

I helped implement the functionality that allows users to have a list of ingredients they have on hand and to search for recipes based on these ingredients.

This project has also taught me how to better work in a team on a medium sized project.

The bottom image shows a sample of what a user's ingredient list may look like. The website is accessible here.

Electronic Drum Kit

Status: Completed

This project spearheaded by the University of Toronto Project Holodeck design team intends to create a working arcade drum kit

As part of this project, I helped interface an Arduino connected to four metal plates and two conductive rods to Unity for the latter to register which rod has hit which plate

I also contributed to working on parsing the drum track from MIDI files into Unity as input for the rhythm game that the arcade will run

I have completed all my assigned sections for this project before it was halted due to the COVID-19 pandemic.

This was my first in-depth experience working with Arduino.

3D Environment Room Scanner

Status: In Progress

This is another project spearheaded by the University of Toronto Project Holodeck design team.

The aim of this project is to create a platform allowing the real-time scanning of an environment into a virtual 3D representation

The completed project will be able to use an HTC Vive controller paired with a LiDAR and a 360 camera to then recreate the environment in Unity

I have helped come up with coordinate transformation functions to map the LiDAR readings into a centralized coordinate system

We are currently able to generate a point cloud, however the coordinate transformations still need tweaking.

This project has been put on hold due to the COVID-19 pandemic.
